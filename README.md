## 🎯 这是什么？

花语嫁接师是一个 **基因模拟 × 园艺收集** 游戏。核心概念：

> 🌹 玫瑰遇到 🌻 向日葵？它们的「基因因子」会随机融合，创造出全新的花种！

每朵花由多个 **原子因子** 定义——花瓣形状、颜色、质地、香气……嫁接时每个因子 50% 概率遗传，因子越多 → 评分越高 → 稀有度越高。

### 稀有度体系

| 等级 | 积分 | 数量 | 代表因子 |
|:--:|:--:|:--:|------|
| <kbd>N</kbd> 普通 | 1 | 35 | 红花·圆瓣·清香·直立茎 |
| <kbd>R</kbd> 稀有 | 5 | 6 | 渐变花色·复瓣·条纹·斑点 |
| <kbd>SR</kbd> 珍贵 | 10 | 5 | 金属光泽·荧光·流星纹·双层花瓣 |
| <kbd>SSR</kbd> 史诗 | 20 | 4 | 水晶瓣·星辉·浮空·三色渐变 |
| <kbd>UR</kbd> 传奇 | 50 | 5 | 🌈虹彩·🐉龙鳞瓣·🌌星空·✨圣光·⏳时幻 |

### 融合规则

```
  7种颜色共存  ──→  虹彩 (UR)       // 七彩流光
  3~6种颜色    ──→  渐变花色 (R)     // 自然过渡
  重瓣 + 单瓣  ──→  复瓣 (R)        // 复杂层叠
  杯状 + 蝶形  ──→  双层花瓣 (SR)    // 蝶杯花冠
  3种+香气     ──→  荧光 (SR)        // 迷魂香
  光滑 + 绒面  ──→  金属光泽 (SR)    // 丝绒光泽
```

## 🚀 本地运行

```bash
git clone git@github.com:gethubws/Flower_Grafting_technique.git
cd Flower_Grafting_technique

# 基础设施
docker-compose up -d              # PG + Redis + MinIO

# 后端
cd server && npm install
npx prisma migrate dev
npm run start:dev                 # :3000

# 前端
cd web && npm install
npm run dev                       # :5173

# AI 网关（可选，需 SD Forge）
cd ai-gateway
source venv/bin/activate
uvicorn main:app --port 8001     # :8001
```
